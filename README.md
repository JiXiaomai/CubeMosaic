# CubeMosaic


将图像转换成魔方拼图，支持文字拼图和二维码拼图，6种转换算法适合不同类型的拼图。Rubik's cube mosaic building software.

抖动模式：适合非人像拼图，使用近似加抖动算法转换颜色，尽量贴近原始颜色，在魔方数量较多时效果最佳；

灰度模式：适合人像拼图，使用灰度转换颜色，体现色彩变化，但不还原真实颜色；

近似模式：适合LOGO、艺术字等字符图案拼图，对图像的色彩还原程度较差。
